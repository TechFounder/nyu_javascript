Objectives

Upon completion of the lecture you should be able to:
  • Understand the concepts of OOP, and how it differs from other programming techniques.
  • Define your own custom objects using object constuctors.
  • Define properties and methods for you custom object.
  • Instantiate and manipulate your own real world objects.
  • Differentiate between Class and Object properties and methods.
