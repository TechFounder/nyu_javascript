Objectives

Upon completion you should be able to:

  • Understand the purpose of CSS
  • Understand the structure of a CSS rule
  • Create an inline CSS rule that applies ot a single element
  • Embed internal CSS that can be applied to entire page
  • Link to external CSS that can be applied to many pages
  • Understand the difference between block and inline elements
  • Understand the purpose for the <div> and <span> tags
  • Create simple CSS selectors
  • Create and apply class and id selectors
  • Create contextual selectors such as descedant, child and sibling selectors
  • Combine various selectors for pin point element targeting
  • Understand the difference between cascading and inheritance rules
