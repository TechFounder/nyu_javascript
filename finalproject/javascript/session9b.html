Objectives

Upon completion you should be able to:

  • Understand what jQuery is
  • Download and properly use jQuery
  • Select one or more element from the DOM
  • Alter element content
  • Add, change and remove element attributes
  • Add, remove and toggle element CSS classes
  • Add and change CSS properties
  • Add, replace and remove elements
  • Use jQuery animation effects such as hide/show, fade and slide
  • Process elements individually within an element set
  • Process elements individually within an array or an object
  • Set and retrieve cookies using the jQuery cookie plugin
