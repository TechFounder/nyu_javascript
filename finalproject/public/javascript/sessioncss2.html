Objectives

Upon completion you should be able to:

  • Apply various text related properties to the web page
  • Apply various font related properties to the text of the web page
  • Understand the element box model including border, margin and padding
  • Use the box model to control element height and width
  • Apply background to various elements
  • Understand the difference between relative and absolute positioning
  • Position elements using either relative, absolute or fixed positions
  • Control Overlapping elements using the z-index property
  • Understand the various length measures including absolute, relative and percent.
